# 🧠 Breast Cancer Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting whether a tumor is **benign or malignant** using Machine Learning algorithms.

The goal is to assist in early detection and improve decision-making in healthcare.

---

## ❗ Problem Statement
Breast cancer is one of the most common cancers worldwide.  
Manual diagnosis can be time-consuming and prone to errors.

👉 This project aims to build a predictive model that can **classify tumors accurately** based on input features.

---

## 💡 Solution
Developed a Machine Learning model that analyzes medical data and predicts cancer type using classification techniques.

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook  

---

## 📊 Dataset
- Breast Cancer Dataset (from sklearn / UCI repository)
- Features include:
  - Radius
  - Texture
  - Perimeter
  - Area
  - Smoothness
  - etc.

---

## ⚙️ Workflow
1. Data Collection & Understanding  
2. Data Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Model Training  
6. Model Evaluation  

---

## 🤖 Models Used
- Logistic Regression  
- Support Vector Machine (SVM)  
- Decision Tree  
- Random Forest  

---

## 📈 Results
- Achieved high accuracy (~90%+)  
- Model successfully classifies tumors as benign/malignant  
- Random Forest performed best among models
